# The Great Blue Heron — Content Hub & Sitemap

A research-backed content library about the Great Blue Heron (*Ardea herodias*), organized so each file can stand alone as a web page. Every page carries its own source citations at the bottom. Built for an educational / nature-nonprofit audience.

## Quick species snapshot

| Fact | Value |
|---|---|
| Scientific name | *Ardea herodias* Linnaeus, 1758 |
| Order / Family | Pelecaniformes / Ardeidae |
| Length | 97–137 cm (38–54 in) |
| Weight | 2.1–2.5 kg typical; up to 3.6 kg (4.6–7.9 lb) |
| Wingspan | 167–201 cm (66–79 in) |
| Global population | ~700,000 breeding birds (Partners in Flight) |
| IUCN status | Least Concern, population increasing (assessed 2020) |
| Average lifespan | ~15 years; banding record 23–25 years |
| Range | North & Central America, Caribbean, Galápagos; vagrant to Europe |

## Editorial guidance for reuse

- Each page is self-contained; internal cross-links use relative paths, so the folder can be dropped into any static site structure.
- Citations use Markdown footnotes local to each page — renumber freely if merging pages.
- Charts live in `images/` and are referenced with relative paths.
- Suggested expansions: state/province-level pages (the sources include Maine, North Carolina, California, Oregon, Washington, BC, and Adirondacks detail), a "heron of the month" seasonal series, and kid-friendly rewrites of pages 02, 08, and 20.
